# 📊 KlierNav CRM & Digital Services Catalog

Plataforma unificada de gestión de leads (CRM) y catálogo de servicios digitales. Desarrollada como herrramienta operativa principal para los vendedores y gestores de **KlierNav Innovations**.

Esta aplicación React de alto rendimiento integra módulos de arrastrar y soltar (Kanban), métricas en tiempo real y componentes optimizados para flujos de venta ágiles.

---

## ✨ Características Principales

- **Tablero Kanban Interactivo:** Gestión visual del pipeline de ventas. Posibilidad de arrastrar *leads* a través de los diversos estados del embudo.
- **Dashboard Analítico:** Gráficos integrados (vía Recharts) para medir el rendimiento de ventas, conversiones y proyecciones MRR (Monthly Recurring Revenue).
- **Catálogo de Servicios Integrado:** Fichas de producto accesibles internamente para cotizar velozmente servicios como "Lead Genie" o "Cloud Mirror for Mails".
- **Sistema de Chat Mockeado:** Interfaz base para comunicación interna o puente con WhatsApp.
- **Autenticación (WIP):** Capa de seguridad preparada para integración con Supabase Auth.

## 🛠️ Stack Tecnológico

- **Frontend:** React 19 + TypeScript + Vite
- **Estilos:** Tailwind CSS
- **Componentes DND:** `@hello-pangea/dnd` para un drag & drop robusto
- **Gráficos:** Recharts
- **Iconos:** Lucide React

## 🚀 Inicio Rápido

1. Instalación de dependencias:
   ```bash
   npm install
   ```
2. Iniciar servidor local:
   ```bash
   npm run dev
   ```
3. Construir para producción:
   ```bash
   npm run build
   ```
